# Tahams • Typography & Campaign Design Guidelines

This document serves as the official typography design guidelines for **Tahams**. It details the **32 curated font pairing configurations** and **4 detail layout styles** matching your brand campaigns, graphic products, packaging, and marketing collateral. All fonts recommended here are available for free via **Google Fonts**.

---

## 🏷️ Brand Metadata & Detail Layout Styles

Use these four layout blocks for secondary visual detailing, product tags, sleeve prints, and side stamps:

### Style 1: Vertical Stamps & Date Stamp Layout
* **Visual Match**: Inspired by the *people come and go* vertical calendar layout.
* **Date Block**: Large, high-contrast numeric display (e.g. `24 / 02`) in **Space Grotesk** or **Inter** (Bold, 800).
* **Vertical Sidebars**: Set using `writing-mode: vertical-rl` in **Inter** or **Montserrat** (Uppercase, tracking `+2px`, size 10px-11px). Ideal for listing collection series, release days, or manufacturer tags.

### Style 2: Narrative Column Block
* **Visual Match**: Inspired by the *people come and go* side-aligned narrative columns.
* **Multi-lingual Accent**: Vertical Asian character text lines (`writing-mode: vertical-rl`) block.
* **Paragraph Block**: Clean, narrow paragraph columns (width: 180px - 220px) in **Plus Jakarta Sans** or **Outfit** (Regular, size 11px-12px, line-height `1.7` for breathing room). Use for fabric story snippets or philosophical quotes.

### Style 3: Wash Tag & Product Specification Layout
* **Visual Match**: Inspired by custom wash tags and technical prints.
* **Spec Grid**: Center-aligned or left-aligned lists in **Space Grotesk** (Uppercase, size 10px, tracked `+1.5px`) with bullet dots (`•`) separating values (e.g., colorways, sizing, cotton weight).
* **Dividers**: Fine 1px borders combined with central circle divider dots for an architectural look.

### Style 4: Technical Specs & Grid Layout
* **Visual Match**: Inspired by the *Amidst the Green* camera specs grid layouts.
* **Red Tag Block**: Bold solid red rectangle badge (e.g. `12.8`) in **Space Grotesk** or **Inter** (Bold, 800) with thin camera description text (`camera : Samsung A10s`) underneath in **Outfit** (Regular, 11px).
* **Symbol Overlays**: Minimal arrows (`➔`, `←`) and serial listings (e.g., `46U5T • 2K25`) in **Space Grotesk** (Uppercase, size 10px).

## 📐 Graphic Slogan Design Hierarchy

When creating graphic t-shirts or campaign assets, designers should lock up text using one of the following two styles:

### Style A: Bold Display Dominant (Default)
* **How it works**: The primary slogan message (e.g. `"Dominate"`) is set in **Font 1** (Large, Bold, All-caps). The supporting text (e.g. `"Born to"`) is set in **Font 2** (Small, Cursive/Script/Italic).
* **Vibe**: High impact, modern, commercial.

### Style B: Cursive / Script Dominant (Vice Versa)
* **How it works**: The primary slogan message is set in **Font 2** (Large Cursive Script, flowing, Title Case). The supporting text is set in **Font 1** (Small, Sans/Serif, All-caps, spaced).
* **Vibe**: Organic, classic, vintage, premium signature.
